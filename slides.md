# 研究背景

Slidev is a slides maker and presenter designed for developers, consist of the following features

- 📝 **研究主题** - 利用强化学习改进遗传算法进行车间调度研究
- 🎨 **研究意义** - 通过强化学习，改善遗传算法易陷入局部最优、收敛速度慢、**参数选择困难**的缺点
- 🧑‍💻 **研究目标** - 利用 Q-learning 等方法，使得遗传算法中的交叉率可以自学习，从而求解车间调度问题
- 🤹 **研究范围** - 作业车间问题(Job Shop Schedule)的最小化时间
- 🎥 **强化学习内容** - Q-learning 算法、SARSA 算法
- 📤 **遗传算法内容** - 选择：轮盘法、锦标赛法；交叉：单点交叉、多点交叉；
- 🛠 **最终目标** - 实现RLGA算法，解决车间调度问题，并与GA算法进行对比
